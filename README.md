# Geodesique

Hobby research into biologically-motivated unsupervised learning on the unit hypersphere. The central idea: a population of templates lives on the hypersphere, and a single mechanism — the Leave-One-Out (LOO) shadow, the collective reconstruction with one template removed — drives both lateral inhibition and synaptic plasticity. Weight updates are geodesic rotations (Rodrigues' formula), so templates never leave the sphere. Two algorithms evolved from this: **GeodesicIMDVv2**, which uses explicit pull/push forces balanced by a single E/I parameter, and **CRG** (Contrast-Residual Geodesic), a cleaner successor that mean-centers the input to extract contrast and uses the pure directional residual as the sole learning signal — no balance parameter, no separate push force. Both are online, parameter-sparse, and have no backpropagation. Built on top of [AxonForge](https://github.com/tagrikli/axonforge), a personal node-graph framework. Evaluated on MNIST (91.7% test accuracy) and Fashion-MNIST (82.6%) via a linear probe — features are learned entirely unsupervised from 100 templates, then a logistic regression is fit on top with labels.
